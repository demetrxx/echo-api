# Aera

## Description

Telegram AI-girlfriend chatbot.

## Installation

```bash
$ pnpm install
```

## Running in dev mode:

1. Run ngrok to get public URL (set to `APP_URL` in `.env` automatically)
```bash
  ngrok.sh
```

2. Run the app
```bash
pnpm run start:dev
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
